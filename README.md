# Imperial NLP Coursework submission

The cleaned dataset after running our proposed EDA method is located at the top directory, named dontpatronizeme_pcl_cleaned.tsv.

The labels, or the Dev set predictions (dev.txt) and the Test set predictions (test.txt) are located in the labels directory.

The model, named as model.safetensors, as well as the notebook containing the implementation of the model, named as nlp_final.ipynb, are both located in the BestModel folder.

As mentioned in the report, our implementation achieved an F1 score of 0.5891 on the development set, with a threshold=0.30, during epoch 4.
